# Implementation Package Template

Purpose:
Give the founder, ChatGPT, and Codex one compact implementation handoff artifact for approved work.

Rules:
- Keep it concise, specific, and implementation-ready.
- Only prepare a full package for work the human is willing to consider for implementation.
- Trello content stays in proposal form until explicit human approval.
- Do not treat the package as a substitute for human judgment.
