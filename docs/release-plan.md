# Release Plan

This release plan describes the path from the current prototype toward NextMove as an AI-powered Career Operating System.

## v0.3 Foundation

- Stabilize Career Vault data.
- Keep job application records simple and inspectable.
- Preserve local-first storage while the workflow is still changing.
- Clarify product identity, roadmap, and UX direction.
- Maintain basic tests for storage and data helpers.

## v0.4 Guided Workflow

- Improve Jobs Applied navigation and page flow.
- Connect Dashboard, Add Job, Job Detail, Fit Analysis, Application Packet, and Tracking into one guided path.
- Add status editing, follow-up dates, and next-action prompts.
- Reduce duplicate typing between job records and generated outputs.

## v0.5 AI Career Coach

- Add controlled AI generation for fit analysis, tailored resume drafts, and cover letter drafts.
- Keep outputs grounded in Career Vault and saved job postings.
- Track prompt version, model name, generated timestamp, and user approval.
- Show reasoning and uncertainty clearly.
- Avoid exaggerated claims.

## v0.6 Interview Coach

- Generate interview prep from job requirements and Career Vault evidence.
- Create proof stories, gap bridges, and questions to ask the employer.
- Help users prepare concise answers for common and role-specific questions.
- Track interview notes and follow-up actions.

## v0.7 Career Intelligence

- Help users identify patterns across jobs, recommendations, outcomes, and interviews.
- Surface which types of roles are strongest, healthiest stretches, or poor fits.
- Provide career positioning insights over time.
- Support longer-term growth planning beyond the current job search.

## v1.0 Public Beta

- Provide a coherent end-to-end experience from saved posting to interview prep.
- Include stable data handling and clear privacy expectations.
- Offer polished UX for core job search workflows.
- Support exportable application materials.
- Prepare for external feedback, onboarding, and public product validation.
