# Glossary

- `Career Brain`: the evolving reusable source of truth for a user's professional story, evidence, preferences, and goals.
- `Profile / Story Bank / Career Vault`: earlier or adjacent terms for the same core source-of-truth concept.
- `Opportunity`: a job posting or role the user is evaluating.
- `Opportunity Workspace`: the job-specific place where fit analysis, outputs, notes, and status come together.
- `Application Studio`: the area that generates job-specific outputs such as resume drafts, cover letters, and related assets.
- `Implementation Package`: the repo artifact that turns planning into a Codex-ready handoff before a GitHub Issue is created.
- `Founder Decision`: a choice that needs explicit founder judgment before implementation proceeds.
- `Repo Backlog`: the docs-managed backlog where ideas, features, process improvements, and debt live until ready for issue creation.
