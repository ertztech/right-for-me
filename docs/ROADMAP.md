# Roadmap

## v0.1 - Job Fit and Stretch Analyzer

Goal: help a job seeker decide whether a role is worth applying to.

- Paste job description
- Add optional background or resume text
- Extract role requirements
- Estimate fit score
- Estimate stretch score
- Show evidence to emphasize
- Show gaps and risks
- Recommend next action

## v0.2 - Application Positioning

Goal: turn the analysis into a practical application strategy.

- Suggested resume angle
- Suggested headline
- Best experiences to foreground
- Keywords to include honestly
- Risks to address directly

## v0.3 - Resume Bullet Assistant

Goal: help the user tailor existing experience without inventing claims.

- Rewrite bullets for the target role
- Preserve truthfulness
- Highlight adjacent strengths
- Explain why each bullet matters

## v0.4 - Cover Letter Drafting

Goal: create concise, specific cover letters grounded in the role and company.

- Company-aware cover letter draft
- Role-specific motivation
- Evidence-backed claims
- Optional tone controls

## v0.5 - Saved Opportunities

Goal: help users compare opportunities over time.

- Save job analyses
- Compare fit and stretch
- Track application status
- Record notes and decisions

## v1.0 - Public Launch

Goal: a coherent, useful tool that someone can use end-to-end.

- Stable analysis workflow
- Saved jobs
- Resume positioning
- Cover letter drafting
- Interview prep starter
- Documentation
- Public demo
