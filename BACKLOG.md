# RightForMe Backlog

This backlog describes product work in plain language. It is meant to be readable by a non-technical user and useful for planning future branches.

## Jobs Applied Backlog

The Jobs Applied pipeline will help a user move from "I found a role" to "I applied and know what to do next." It should connect job postings, fit decisions, tailored documents, application status, and follow-up actions without inventing experience.

### MVP

- Add a job posting
  - As a job seeker, I want to paste or save a job posting so I can evaluate it later without losing the original text.
- Score job fit
  - As a job seeker, I want the app to score my fit for a saved job so I can decide whether it is worth pursuing.
- Save fit analysis
  - As a job seeker, I want to save the fit score, recommendation, evidence, and gaps so I can return to the decision later.
- Generate tailored resume
  - As a job seeker, I want to generate a resume tailored to a saved job using my Career Vault so I can apply with relevant evidence.
- Generate cover letter
  - As a job seeker, I want to generate a cover letter for a saved job using my Career Vault so I can explain why I am interested.
- Mark job as applied
  - As a job seeker, I want to mark a job as Applied so I can separate active applications from roles I am still reviewing.
- Store company, role, URL, and notes
  - As a job seeker, I want to save the company, role title, job URL, and notes so the application record has the basic context I need.

### Next

- Track application status
  - As a job seeker, I want to update a job status from Found through Applied, Interviewing, Rejected, Offer, or Closed so I can see where each opportunity stands.
- Add follow-up reminders
  - As a job seeker, I want to set a follow-up date so I know when to check in or take the next action.
- Store salary and location details
  - As a job seeker, I want to save salary range, location, and work arrangement so I can compare opportunities more honestly.
- Store generated output paths
  - As a job seeker, I want the record to remember which tailored resume and cover letter belong to a job so I can find them later.
- Add interview prep notes
  - As a job seeker, I want to save likely questions, evidence to emphasize, and gaps to prepare for before an interview.

### Later

- Export application packet as Markdown and PDF
  - As a job seeker, I want to export a complete application packet so I can archive or share my materials outside the app.
- Application follow-up dashboard
  - As a job seeker, I want to see overdue follow-ups and active opportunities in one place so I can manage the search without a spreadsheet.
- Status history
  - As a job seeker, I want to see when each application moved between statuses so I can understand my job search activity over time.
- Source comparison
  - As a job seeker, I want to compare roles by source, salary, location, and recommendation so I can learn which opportunities are most promising.
- Automation experiments
  - As a job seeker, I want optional reminders and next-step suggestions once the core workflow is stable.
